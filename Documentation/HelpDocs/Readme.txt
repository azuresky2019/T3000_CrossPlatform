T3SoftwareManual.PDF is the user guide for the original DOS version T3000 program. It includes info on how to run and program a system. 

T3TechncalManual.PDF is more related to the original hardware whcih ran on an embedded PC. It is included for reference but not very useful since the hardware has changed so much. The latest hardware is documented in the Windows version of this project: 
https://github.com/temcocontrols/T3000_Building_Automation_System/tree/master/Documentation

RPi_Compute_Module_Instructions.docx is a complete installation instruction for the Raspberry Pi compute module.  It includes instructions on flashing the image file to the RPi and installing everything necessary to use the T3000 GUI on a T3 Router.
