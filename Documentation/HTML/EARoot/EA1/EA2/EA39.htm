Loads a list of Programs Control Points into a gridview, allowing user to interact with any row.<br/><br/>A ProgramPoint, is a control point, a subheader of information related to a ProgramCode. It contains: Label, Name, Lenght and Index.<br/><br/>Each index in a control point list acts as universal key to locate in counterpart section, the corresponding control code or value.<br/><br/>So, ProgramPoint[0] contains a subheader of information for the real code stored in ProgramCode[0].<p/>
