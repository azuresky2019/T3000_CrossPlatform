Main class of Irony - .NET Compiler Construction Kit<br/>Roman Ivantsov, 4 Jan 2008<br/><br/><br/>Irony is a development kit for implementing languages on the .NET platform. It uses the flexibility and power of the C# language and .NET Framework 3.5 to implement a completely new and streamlined technology of compiler construction.<br/><br/>Unlike most existing yacc/lex-style solutions, Irony does not employ any scanner or parser code generation from grammar specifications written in a specialized meta-language. In Irony, the target language grammar is coded directly in C# using operator overloading to express grammar constructs. Irony's scanner and parser modules use the grammar encoded as a C# class to control the parsing process.<br/><br/>The Irony project is hosted on CodePlex.<br/><br/>Irony brings several principal innovations into the field of compiler construction. Like many parser-building tools in use today, Irony produces a working parser from grammar specifications. However, unlike the existing parser builders, Irony does not use a separate meta-language for encoding the target grammar. In Irony, the grammar is encoded directly in C# using BNF-like expressions over grammatical elements represented by .NET objects. Additionally, no code generation is employed - Irony's universal LALR parser uses the information derived from C#-encoded grammar to control the parsing process.<br/><br/>More info and wikis:<br/><br/>Understanding what is Irony .NET Compiler Construction Kit<br/><a href="https://www.codeproject.com/Articles/22650/Irony-NET-Compiler-Construction-Kit"><font color="#0000ff"><u>https://www.codeproject.com/Articles/22650/Irony-NET-Compiler-Construction-Kit</u></font></a> <br/><br/>Basics of Grammars, Parsing and AST.<br/><a href="https://en.wikibooks.org/wiki/Irony_-_Language_Implementation_Kit"><font color="#0000ff"><u>https://en.wikibooks.org/wiki/Irony_-_Language_Implementation_Kit</u></font></a> <br/><br/>A Brief Article and Sample for Irony Grammar Explorer (for debugging )<br/><a href="http://charliedigital.com/2010/10/01/irony-net-language-implementation-kit/"><font color="#0000ff"><u>http://charliedigital.com/2010/10/01/irony-net-language-implementation-kit/</u></font></a> <br/><br/>Spanish source: Creacion de Gramaticas en Irony<br/>Elaborado por estudiantes del la Universidad de San Carlos de Guatemala de la carrera de Ingenieria en Ciencias y Sistemas<br/>A course on Irony for dummies<br/><a href="http://compi2irony.blogspot.com/"><font color="#0000ff"><u>http://compi2irony.blogspot.com/</u></font></a> <br/><br/>Other<br/><a href="https://www.hanselman.com/blog/TheWeeklySourceCode59AnOpenSourceTreasureIronyNETLanguageImplementationKit.aspx"><font color="#0000ff"><u>https://www.hanselman.com/blog/TheWeeklySourceCode59AnOpenSourceTreasureIronyNETLanguageImplementationKit.aspx</u></font></a> <br/><br/><br/><b>NOTE: T3000 CROSSPLATFORM uses a nuget's package as the most recently updated form of Irony.</b><p/>
